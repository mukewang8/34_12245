<h3>课程介绍：</h3>
<p>本課程著重在 JavaScript 的核心運作觀念，從直譯器的運作深入探討至 JavaScript 的各種運作模式，包含 詞法作用域、提升、非同步、運算子、轉型、物件參考、原型練、建構式等觀念。</p>
<p>課程中除了提供動態簡報說明外，還會提供大量程式碼介紹，並另有許多精彩測驗題，不單能輕鬆地理解相關知識，還能確保學習觀念正確。</p>
